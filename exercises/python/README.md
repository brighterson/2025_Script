# Python Script Practice Exercises

本目录包含一系列Python脚本练习题，帮助你提升Python编程技能。

## 练习题列表

### 1. 基础练习 (basics/)
- 数据类型和变量
- 控制流程
- 函数和模块
- 异常处理

### 2. 文件操作 (file_operations/)
- 文件读写
- CSV和JSON处理
- 文件系统操作

### 3. 数据处理 (data_processing/)
- 列表和字典操作
- 字符串处理
- 正则表达式

### 4. 实用工具 (utilities/)
- 命令行工具开发
- 日志处理
- 自动化脚本

## 使用方法

1. 进入对应的练习目录
2. 阅读 `exercise.md` 了解题目要求
3. 编写Python脚本完成练习
4. 运行测试脚本验证答案
5. 查看 `solution.py` 参考答案

## 难度说明

- ⭐ 初级：适合Python初学者
- ⭐⭐ 中级：需要掌握基本Python语法
- ⭐⭐⭐ 高级：需要综合运用多种Python技巧
