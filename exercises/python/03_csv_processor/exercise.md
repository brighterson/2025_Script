# 练习题 3: CSV数据处理器

**难度**: ⭐⭐

## 题目描述

编写一个Python脚本，处理CSV格式的销售数据：

1. 读取CSV文件（包含：日期、产品、数量、单价）
2. 计算每个产品的总销售额
3. 找出销售额最高的产品
4. 计算所有产品的总销售额
5. 将统计结果输出到新的CSV文件

## 示例CSV格式 (sales.csv)

```csv
日期,产品,数量,单价
2025-01-01,苹果,10,5.0
2025-01-01,香蕉,15,3.0
2025-01-02,苹果,8,5.0
2025-01-02,橙子,12,4.0
2025-01-03,香蕉,20,3.0
```

## 示例输出

```bash
$ python csv_processor.py sales.csv

销售统计报告
=================================
产品销售排行:
  1. 香蕉: ¥105.00
  2. 苹果: ¥90.00
  3. 橙子: ¥48.00

总销售额: ¥243.00

统计结果已保存到: sales_report.csv
```

## 要求

- 脚本名称：`csv_processor.py`
- 使用 `csv` 模块处理CSV文件
- 生成 `sales_report.csv` 包含统计结果
- 处理可能的错误（文件不存在、格式错误等）

## 提示

- 使用 `csv.DictReader` 读取CSV
- 使用字典存储每个产品的销售额
- 使用 `sorted()` 对产品按销售额排序
- 数量 × 单价 = 销售额
