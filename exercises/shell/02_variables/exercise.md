# 练习题 2: 变量和计算器

**难度**: ⭐⭐

## 题目描述

编写一个Shell脚本，实现一个简单的计算器：

1. 接受三个命令行参数：数字1、运算符、数字2
2. 支持的运算符：+ (加)、- (减)、\* (乘)、/ (除)
3. 输出计算结果
4. 如果参数不足或运算符不支持，输出错误信息

## 示例

```bash
$ ./calculator.sh 10 + 5
15

$ ./calculator.sh 20 - 8
12

$ ./calculator.sh 6 \* 7
42

$ ./calculator.sh 100 / 4
25

$ ./calculator.sh 10
错误: 需要三个参数 (数字1 运算符 数字2)
```

## 要求

- 脚本名称：`calculator.sh`
- 需要有执行权限
- 使用 `case` 语句处理不同的运算符
- 使用 `$(( ))` 进行算术运算

## 提示

- 使用 `$#` 检查参数数量
- 除法运算要注意除数不能为0
- Shell中乘号需要转义 `\*`
