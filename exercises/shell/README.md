# Shell Script Practice Exercises

本目录包含一系列Shell脚本练习题，帮助你提升Shell编程技能。

## 练习题列表

### 1. 基础练习 (basics/)
- 变量和字符串操作
- 条件判断
- 循环结构
- 函数定义

### 2. 文件操作 (file_operations/)
- 文件读写
- 目录遍历
- 文件权限管理
- 批量文件处理

### 3. 文本处理 (text_processing/)
- grep、sed、awk使用
- 日志分析
- 文本格式转换

### 4. 系统管理 (system_admin/)
- 进程管理
- 系统监控
- 自动化任务

## 使用方法

1. 进入对应的练习目录
2. 阅读 `exercise.md` 了解题目要求
3. 编写脚本完成练习
4. 运行测试脚本验证答案
5. 查看 `solution.sh` 参考答案

## 难度说明

- ⭐ 初级：适合Shell脚本初学者
- ⭐⭐ 中级：需要掌握基本Shell语法
- ⭐⭐⭐ 高级：需要综合运用多种Shell技巧
