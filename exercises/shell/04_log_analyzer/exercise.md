# 练习题 4: 日志分析器

**难度**: ⭐⭐⭐

## 题目描述

编写一个Shell脚本，分析Web服务器访问日志：

1. 读取日志文件（每行格式：IP地址 时间戳 HTTP方法 URL 状态码）
2. 统计总访问次数
3. 统计不同IP地址的访问次数（显示前5名）
4. 统计HTTP状态码分布
5. 找出访问次数最多的URL（前3名）

## 示例日志格式

```
192.168.1.1 2025-01-13T10:00:01 GET /index.html 200
192.168.1.2 2025-01-13T10:00:05 POST /api/login 200
192.168.1.1 2025-01-13T10:00:10 GET /about.html 200
192.168.1.3 2025-01-13T10:00:15 GET /index.html 404
```

## 示例输出

```bash
$ ./log_analyzer.sh access.log
总访问次数: 150

访问次数最多的IP (前5名):
  45 192.168.1.1
  32 192.168.1.2
  28 192.168.1.3

HTTP状态码分布:
 120 200
  20 404
  10 500

访问最多的URL (前3名):
  50 /index.html
  30 /about.html
  25 /api/login
```

## 要求

- 脚本名称：`log_analyzer.sh`
- 使用 `awk`、`sort`、`uniq` 等文本处理工具
- 输出格式清晰易读

## 提示

- 使用 `awk '{print $1}'` 提取第一列
- 使用 `sort | uniq -c` 统计重复项
- 使用 `sort -rn` 按数字降序排序
- 使用 `head -n 5` 获取前5行
