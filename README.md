# 2025_Script

一个提供Shell脚本和Python脚本练习题的学习库。

Created in 2025, to learn script language like shell or python.

## 📚 项目简介

本项目为学习Shell和Python脚本编程的初学者和进阶者提供系统化的练习题。每个练习都包含详细的题目描述、参考答案和自动化测试脚本。

## 🎯 特性

- ✅ **系统化学习路径**：从基础到高级，循序渐进
- ✅ **完整的练习题**：涵盖Shell和Python常用场景
- ✅ **自动化测试**：每个练习都有测试脚本验证答案
- ✅ **参考答案**：提供高质量的解决方案供学习
- ✅ **中文文档**：完全中文化，易于理解

## 📂 内容结构

```
exercises/
├── shell/          # Shell脚本练习 (4个练习题)
│   ├── 01_hello_world/         # ⭐ 基础：Hello World
│   ├── 02_variables/           # ⭐⭐ 中级：变量和计算器
│   ├── 03_file_backup/         # ⭐⭐ 中级：文件备份工具
│   └── 04_log_analyzer/        # ⭐⭐⭐ 高级：日志分析器
│
└── python/         # Python脚本练习 (4个练习题)
    ├── 01_fizzbuzz/            # ⭐ 基础：FizzBuzz
    ├── 02_file_organizer/      # ⭐⭐ 中级：文件整理工具
    ├── 03_csv_processor/       # ⭐⭐ 中级：CSV数据处理器
    └── 04_web_scraper/         # ⭐⭐⭐ 高级：API客户端
```

## 🚀 快速开始

### 克隆项目

```bash
git clone https://github.com/brighterson/2025_Script.git
cd 2025_Script
```

### 开始练习

#### Shell脚本练习示例：

```bash
# 进入第一个练习
cd exercises/shell/01_hello_world

# 查看题目
cat exercise.md

# 编写你的代码...

# 运行测试
chmod +x test.sh
./test.sh

# 查看参考答案
cat solution.sh
```

#### Python脚本练习示例：

```bash
# 进入第一个练习
cd exercises/python/01_fizzbuzz

# 查看题目
cat exercise.md

# 编写你的代码...

# 运行测试
python3 test.py

# 查看参考答案
cat solution.py
```
